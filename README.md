#  🤖 *POP_AI*

This Final project for coding3 is based on the image generation model StyleGAN2. The initial idea came from my desire to use AI-based image generation models to create stylized posters—enabling the AI to generate visual for poster designs. I plan to apply this model to my graduation project, aiming to generate personalized visual effects based on individual users. This assignment served as an early-stage exploration of that concept, where I experimented with using a machine learning model for image generation and trained it with a dataset in a fixed visual style.
